# SQL Relational Database Project

This project demonstrates the design and implementation of a relational database using SQL Server.

## Database Structure
The database contains the following tables:

- Employee
- Customer
- Orders
- Product
- Order_Details

## Features
- Primary and Foreign Key relationships
- Many-to-Many relationship using a bridge table
- Data integrity using constraints
- Advanced SQL queries using JOIN, GROUP BY, HAVING, and Subqueries

## Files

schema.sql → Database structure  
data.sql → Sample data  
queries.sql → Analytical queries
